# Security Policy

Please do not report security issues in public GitHub issues.

## Reporting

- Email: <security@guardguide.de>
- Alternative: GitHub Security Advisories for this repository

## Scope

This repository contains the public `guardguide.de` website. Reports should
focus on the static site, deployment surface, or repository automation that
belongs to this repo.
